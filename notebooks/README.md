# Notebooks

This folder keeps exploratory notebooks separate from the reusable `brainsurf`
Python package code.

## Structure

- `demos/`
  - `demo.ipynb` - general BrainSurf usage demo.
- `cognitive/`
  - `brainSurf_Cognitive.ipynb` - cognitive EEG feature workflow.
- `meditation/`
  - `MeditatorNonMeditatorComparision.ipynb` - meditator vs non-meditator comparison.
  - `PreMeditationPostMeditationComparision.ipynb` - pre/post meditation analysis.
- `visualization/`
  - `visualizer.ipynb` - plotting and visualization examples.
- `mff/`
  - `parietalAnalysis.ipynb` - MFF parietal analysis workflow.
  - `svyasa.ipynb` - MFF/SVYASA analysis workflow.

## Notes

Many notebooks currently contain absolute local file paths and saved execution
outputs. Review and clean those before using the notebooks as reproducible demos
or examples.
